## Roxio Toast for Mac — Complete Platform Guide

<a href="#">Toast Mac</a> from Roxio is the Mac's definitive disc burning application. Since
its introduction in 1991 — a year before the first Macintosh CD-ROM drive — Toast has been
the disc production tool that Mac users rely on for tasks that go beyond the basic burning
capabilities built into macOS. <a href="#">Roxio Toast Titanium</a> in its current form covers
not just disc burning but the complete workflow around physical media production and video format
management.

While macOS includes disc burning through Finder for basic data discs, <a href="#">Toast Titanium
Mac</a> provides the features that Finder's burning lacks: Video DVD with navigation menus, Blu-ray
production, audio CD with gapless playback and CD-TEXT, disc spanning for large archives, disc

## Data Archival with Disc Spanning

One of <a href="#">Roxio Toast Mac</a>'s most important professional features is disc spanning —
the ability to distribute a large archive across multiple discs automatically when the content
exceeds a single disc's capacity:

**Automatic spanning**: Select a folder of files larger than a single DVD's 4.7 GB capacity.
Toast analyzes the content, determines the optimal distribution across multiple discs, and
burns each disc in sequence, prompting for a blank disc after each disc completes.

**Disc labeling**: Each spanned disc is labeled with sequential numbering automatically.
Toast tracks which files are on which disc in an index stored on the first disc.

**Verification**: After burning, Toast verifies each disc against the original files to confirm
burn integrity. Verification catches and reports any burn errors before the original files
are archived.

Disc spanning makes Toast essential for archival workflows where entire project folders,
photo collections, and video libraries exceed single-disc capacity. Burning a 20 GB project
archive to five DVDs with automatic spanning takes a few button clicks and disc swaps.

## Video DVD Production in Detail

<a href="#">Toast Titanium</a> Video DVD production goes substantially beyond Finder's disc
burning in creating proper Video DVD format discs:

### Encoding and Compatibility

Video DVDs require MPEG-2 encoded video at specific bitrates and resolutions. Source files in
MP4, MOV, AVI, MKV, and other formats must be converted to MPEG-2 during the burning process.
<a href="#">Toast Mac</a> handles this conversion automatically — add source files in any
format and Toast converts during the burn.

NTSC and PAL standard selection ensures the disc plays correctly in the destination geographic
market. USA/Canada releases use NTSC (29.97fps); European releases use PAL (25fps). Toast
handles the frame rate conversion when the source footage is in the other format.

### Menu Design

The navigation menu that appears when a Video DVD is inserted in a player is designed within
Toast using its menu template system:

Select a menu theme from the library of included templates. Assign video clips to chapters
within each title. Set custom thumbnail frames for each chapter button. Preview the complete
menu navigation before burning to verify the disc structure and button navigation behavior.

The menu templates include matching main menu, chapter menu, and scene selection screen layouts
that create professional-looking navigation without design work.

### Dual-Layer DVD

Standard single-layer DVDs hold 4.7 GB. Dual-layer DVD-R DL discs hold 8.5 GB — nearly double —
allowing longer video content without quality reduction. <a href="#">Toast Titanium Mac</a>
burns to dual-layer discs when inserted, automatically setting the layer break point at a
scene transition for smooth playback.

## Audio Production Features

<a href="#">Toast CD burner</a> capabilities include a professional-level audio CD production
environment:

### CD Mastering Features

**Pre-gap editing**: The silence before each track on an audio CD is configurable. Standard
2-second gaps, gapless transitions, and custom gap lengths are configurable per track.

**Cross-fades**: Apply a cross-fade transition between consecutive tracks so one track fades
out as the next fades in, creating continuous mix-style playback on the finished disc.

**Index markers**: Add sub-track index markers to tracks for compatible CD players that support
sub-track navigation.

**Peak normalization**: Automatically normalize the peak level of each track to a consistent
level, preventing volume jumps between tracks from different sources.

### Vinyl and Tape Digitization

<a href="#">Roxio Toast software</a> analog capture records from turntables and tape decks:

Connect a turntable or cassette deck through a USB audio interface or the Mac's audio input.
Toast's audio capture interface monitors the incoming level, records the analog source, and
applies optional noise reduction to remove turntable rumble, tape hiss, and surface noise.

The automatic track detection analyzes silence between songs to split the continuous recording
into individual track files. Manual split points override the automatic detection for recordings
where silence detection is unreliable.
